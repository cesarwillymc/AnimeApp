## Configuration: 

### Local.properties
Add  this line
```
url_graphql=https://rickandmortyapi.com/graphql
name_database=your_name_database
split_io_key = your_split_io_key
feature_flag= anime_app_split
```

### Gradle Java

Go to **Gradle JDK** and change to **Java17**

![Java](image/gradle_j17.png)
